E-dado

Flutter Dart Casual App Mobile

Visão Geral

E-dado é um aplicativo móvel casual, desenvolvido em Flutter e Dart. O objetivo do projeto é criar um aplicativo que simule um lançamento de dados.

Desenvolvi esse projeto para auxiliar os usuários quando não disparem de dados reais para algum jogo de tabuleiro, RPG ou outros. Ele pode ser usado de forma bem variada tanto em número de dados ou dos valores de cada dado. Além disso possui histórico dos lançamentos feitos e somatório de cada jogada.

Funcionalidades & Detalhes

Rolagem de dados

Faz a rolagem de dados ao clicar de um botão, tendo uma boa aleatoriedade dos números que vão sair.

Configuração de dados

Pode-se configurar a quantidade de dados a serem rolados, além do valor máximo dos dados. Também se pode configurar o tipo de valor que será mostrado.

Histórico de lançamentos

Mostra o histórico de lançamentos feitos, com o somatório de cada jogada.

Preferências

Pode-se configurar se o lançar dos dados gerará algum som ou não.
