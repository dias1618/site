Política de Privacidade

Última atualização: Janeiro 2024

1. Introdução

Bem-vindo ao E-Dado! Este aplicativo foi desenvolvido para proporcionar uma experiência simples e divertida de rolagem de dados virtual. Esta política de privacidade explica como tratamos suas informações ao usar nosso aplicativo.

2. Coleta de Dados

O E-Dado foi projetado para funcionar com o mínimo de dados possível. Coletamos apenas:

  • Configurações do Aplicativo: Suas preferências de som e configurações de dados são armazenadas localmente no seu dispositivo.
  • Histórico de Rolagens: O registro das suas rolagens de dados é armazenado apenas localmente no seu dispositivo.

3. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Manter suas preferências de som
  • Exibir o histórico de rolagens
  • Melhorar a experiência do usuário

4. Armazenamento de Dados

  • Todos os dados são armazenados localmente no seu dispositivo
  • Não realizamos nenhum envio de dados para servidores externos
  • Não compartilhamos nenhuma informação com terceiros

5. Permissões do Aplicativo

O E-Dado requer apenas as seguintes permissões:

  • Internet: Para abrir links externos (política de privacidade e termos de uso)
  • Áudio: Para reproduzir sons durante a rolagem dos dados

6. Dados de Crianças

O E-Dado não coleta intencionalmente dados de crianças menores de 13 anos.

7. Alterações na Política de Privacidade

Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos verificar esta página regularmente para quaisquer alterações.

8. Seus Direitos

Você tem o direito de:

  • Acessar seus dados armazenados localmente
  • Apagar o histórico de rolagens
  • Redefinir as configurações do aplicativo

9. Exclusão de Dados

Para excluir seus dados:

  1. Apague o histórico de rolagens através da interface do aplicativo
  2. Desinstale o aplicativo para remover todas as configurações locais

10. Segurança

Seus dados são armazenados com segurança apenas no seu dispositivo. Não temos acesso a nenhuma informação pessoal.

11. Cookies e Tecnologias Similares

O E-Dado não utiliza cookies ou tecnologias similares de rastreamento.

12. Consentimento

Ao usar o E-Dado, você concorda com esta Política de Privacidade.
